Step Three: Finding Linear Factors Learn More
Please remember that "linear factors" are of the form ax + b, where a and b are constants and the variable has a power of 1. We have learned earlier that if a number r is a Zero of the polynomial function, then is a linear factor of the function. Knowing this we can write any polynomial function as a product of linear factors by simply replacing the r in the linear factor, in turn, with every Zero and then connecting all of the linear factors with multiplication.
